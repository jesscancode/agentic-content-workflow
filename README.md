# Agentic Content System

A reusable framework for running a content operation with AI agents, deterministic automation, and human decision gates.

**→ [View the framework](https://jesscancode.github.io/agentic-content-workflow/agentic-content-system.html)**

## What it is

A single-page reference for the [AI product-framework harness pattern](https://jesscancode.github.io/ai-product-framework/) applied to content:

- **Agents** — only where judgment is needed (research, write, critique, compliance, brand, analyse)
- **Automation** — deterministic scripts for the repeatable steps (schema, linking, metadata, publish, CI/CD)
- **Humans** — own strategy and the five approval gates

It ships with the pipeline (11 layers), the harness services (Content Store, Conductor, gates), governance principles, and a full RACI matrix. Roles, tools, and domain specifics are placeholders — rename them to match your team.

## Deploying

Served via GitHub Pages from the repo root. In **Settings → Pages**, set the source to the `main` branch (root), then open the link above.

## Local viewing

Open `agentic-content-system.html` directly in a browser — it's self-contained, no build step.
